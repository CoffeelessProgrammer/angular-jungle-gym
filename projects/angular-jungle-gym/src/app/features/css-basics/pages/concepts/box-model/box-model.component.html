The Box Model

Block-level Elements

  • Block-level elements consist of a box made of 4 parts:
    • Content
    • Padding
    • Border
    • Margin
      • Margins collapse, i.e. when margins are adjacent, larger value wins.
  • Block-level elements take up the full width of their container by default

The Box Model

Box Model Properties

  • padding: t r b l;
  • margin: t r b l;
  • border: 4px solid #2C3E50; (shorthand)
    • border-width
    • border-style
    • border-color
  • width - % is relative to container
  • height - % is relative to container
  • box-sizing - determines which parts of the box model are used for width & height calculation of block-level elements
    • content-box (default)
    • padding-box
    • border-box (recommended)
  • display - determines positional behavior of element
    • block
    • inline
    • inline-block
    • none

visibility: hidden vs. display: none

  • visibility: hidden
    • Element is still part of document flow and takes its usual space
  • display: none
    • Element is no longer part of document flow, i.e. takes no space
  • In both cases, the element is still part of the DOM.
