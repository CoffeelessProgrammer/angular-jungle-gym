Container & Axes

  • Put a container around the elements to control, and set display property on container to flex.
  • Flexbox only applies to direct children of the container.
  • flex-flow – shorthand for flex-direction & flex-wrap
    • flex-direction determines the direction of the Main and Cross axes.
      • Values: row, column, row-reverse, column-reverse
      • Main axis is the direction of flex flow, i.e. horizontal if row, else vertical
      • Cross axis is perpendicular to the Main axis
    • flex-wrap determines how to handle overflow, i.e. whether flex items should wrap on Cross axis
      • Values: nowrap, wrap, wrap-reverse
  • justify-content – children spacing along Main axis
    • Values: flex-start, flex-end, center, space-between, space-around
  • align-items – children spacing along Cross axis
    • Values: flex-start, flex-end, center
  • align-content – spacing of multiple lines, i.e. w/ flex-wrap
    • Values: flex-start, flex-end, center, space-between, space-around

Flex Item Manipulation

  • flex – [flex-grow] [flex-shrink] [flex-basis]; (shorthand)
    • flex-grow – ratio at which to distribute leftover space in container b/w items
    • flex-shrink – ratio at which items shrink on smaller screens; 0 disables shrinking
    • flex-basis – similar to width
  • order
  • align-self – override container-level align-items for specific child item
