Basics

Data Binding

  • Event Binding

    <button (click)="expression()">...</button>
<input (input)="expression()">

  • Property Binding

    <a [href]="expression()">...</a>

  • String Interpolation

    <li>{{ expression() }}</li>

  • Two-Way Binding

    <input [(ngModel)]="variable">

Directives

  • *ngIf

    <div *ngIf="condition; else #identifier">...</div>
<ng-template #identifier>...</ng-template>

  • *ngFor

    <li *ngFor="let link of usefulLinks"></li>

  • [ngStyle]

    <li [ngStyle]="{ 'css-property': value() }"></li>

  • [ngClass]

    <li [ngClass]="{ className: condition() }"></li>
<li [ngClass]="{ 'class-name': condition() }"></li>
